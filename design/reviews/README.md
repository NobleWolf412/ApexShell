Builder build (`persona-builder-v1`) — the findings and verdicts captured as
that feature was built, slice by slice. They are **not** live specs and are not
maintained against the current code.

For what shipped, see `design/persona-builder-v1.md` (the spec, marked
implemented) and floorplan.md § Extensions / the persona system.
